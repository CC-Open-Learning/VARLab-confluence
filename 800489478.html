VARLab : VARnival/VARNiverse

Created by Nick Harrison, last modified by Hayden Auterhoff on Mar 19, 2024

VARNiverse is a sandbox project designed for an in-between-level project compared to the current onboarding and code jam projects of the adventure game roll-a-ball. this project has a main scene with built-in functionality and a way to transition from this main scene to a scene the developer has set up to do either their onboarding project or jam project.

For VR functionality in VARniverse, check out these requirements and steps

Unity 2022.3.21f1 is required for the VR packages in VARnival

Note: there will be a pop-up about using the project files or downloading the packages from online. Select the “yes, download” option as this will download these packages, as these are important for the VR functionality.

insert pop-up if we get a screenshot

NOTE:
Any project added to VARniverse MUST use the built-in render pipeline meaning projects using any other pipeline (URP, UDRP) will not work. Sometimes these projects can be converted to the built-in renderer but this is not always an easy task.

In addition, VARniverse is currently only setup to support 3D games and projects

Lastly, multiscene projects are supported by VARniverse as long as they follow the above criteria
