Step 1 - Starting Point and Login

The installation of Photon Fusion doesn’t involve the package manager in Unity, as it is a downloaded SDK package from their website. To start the installation process, click on the link.

https://doc.photonengine.com/fusion/current/getting-started/fusion-intro

From this page, click on the SDK & Release Notes tab.

You need to create an account or sign in. Otherwise, you won’t be able to download package.

Step 2 - SDK Download

Download the latest Fusion SDK package, and save the file in a known area

Step 3 - Installing into Unity

Open up the Unity project and drag the file into the project folder (either Assets or Packages).

The Import Unity Package will open after placing the SDK package into the Unity package. Select Import without unchecking any of the assemblies.

Step 4 - Photon Fusion Setup

After the import is complete, the Photon Fusion Hub window should open automatically.

If not, select the Fusion tab on the top of the project and select Fusion Hub (or press Alt+F)

With the Photon Fusion Hub, you will be given additional information about how to set up Fusion, along with Tutorials and Samples, as well as some Documentation.

However, a Fusion App Id is required for networking in the Unity project. The Fusion Hub provides information on where to acquire the Id:

  • Open the Photon Dashboard (Login is required)

  • Select an existing Fusion App Id, or create a new one

  • Copy the App Id and paste into the field on the Photon Hub

With the Fusion App ID copied, paste it into the Photon Fusion Hub and you are ready to go!
