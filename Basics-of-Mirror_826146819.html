Introduction

Mirror hides a lot of the lower-level networking things from you while showing us only the high-level aspects of it. Things such as sending data to another client and the architecture structure are hidden from us, whereas something simple as setting an IP address to connect to the server is something we provide. It is a Black box to us; we give it what it needs to work, and it does it all for us without needing to change anything internally.

Since there is a lot of information, everything will be on its own separate page with all the links on this page.
