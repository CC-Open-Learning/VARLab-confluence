VARLab : Advanced Git Techniques

Created by Jordan Robinson, last modified by David Inglis on Mar 28, 2023

In this section, we’ll cover some advanced Git techniques that come in handy for keeping a clean Git history and in times when things go sideways.

HERE BE DRAGONS

Most of the techniques in this section involve rewriting Git history and can involve destructive operations. If it’s your first time doing any of these operations make sure someone experienced with them is there to help!
