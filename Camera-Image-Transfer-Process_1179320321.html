VARLab : Camera Image Transfer Process

Created by User 96d6e, last modified on Jul 16, 2024

When Transferring the Photos…

Ensure that there will be NO interruption in the picture transfer process as the computer could either corrupt the image(s) or delete them entirely.

So please verify that the computer/laptop will not restart or die while transferring the photos (plug it in to the charger to be safe).

It might also be wise to not have programs, applications or other processes running that might take a significant toll on the laptop.

IMPORTANT: Do not delete photos from the camera until it is confirmed that all photos transferred/downloaded properly.

  • This is a backup in case, as you can redownload the photos from the camera again, but they must exist in order to do so.

Adobe Bridge Workflow →

Once you are done capturing the subject it is now time to transfer the photos to the computer effectively. There is a general workflow provided below that is continuously being worked on. 

The Steps

Step 1.

Hook up the camera to the computer with the correct cord (most likely micro USB).

Step 2.

Open up the Adobe Bridge app and navigate to the “Get Photos from Camera” window.

Step 3.

A window should pop up (as a result of the double click) that looks like this:

A dropdown is demonstrated that provides options for which camera you would like to import from. There is the chance that the application hasn’t had time to catch up with the camera being plugged in and you might need to select <Refresh List> to update the list and find the camera. 

This window also provides some extra settings to optimize the importation process. I typically always use the Create Subfolder(s) by Shot Date (yyyymmdd) as it makes it easy to locate images that were shot on different dates. Folders can always be renamed to have the subject inform of the date to make it easier to locate as well (ex: rollTape20240530). Once the media has been selected (signified by the check marks beside the date and number of each photo) click the “Get Media” button to start the retrieval process. It might be beneficial to write down or remember the number of photos that were selected for future verification. 

The retrieval process might take some time…

Note: I typically won’t select “Delete Original Files” in this window as I want to make sure the files were downloaded properly before deleting them from the camera.

Step 4. 

Double check that the photos were retrieved successfully by locating the folder that was created and doing a general look through of the photos. You can also be precise by recalling the number of photos that were selected to retrieve from the camera and verifying that the number is the same as the number of photos in the folder.

Step 5. 

The next step is to convert the RAW data photos to PNG or TIFF so that they can be successfully imported into Microsoft Agisoft. First click on the “Custom Export” button on the lower left side panel. This command can also be run by clicking the File tab on the top left of the main navbar, and then navigating down to Export → Custom Export.

Step 6.

Create a new preset using the Custom Export Window.

Step 6.1

The following windows will show the desired settings that should be used (on this machine it is saved under RAW to TIFF VARLab).

Match the settings to the settings above, the only changed settings should be highlighted in green but double check just in case.

This is a screenshot of the window a bit further down. Match the settings to the settings above, the only changed settings should be highlighted in green but double check just in case.

This is a screenshot of the window a bit further down, it should be the last of the settings in the Preset/Custom Export Window. Match the settings to the settings above, the only changed settings should be highlighted in green but double check just in case. MAKE SURE TO INCLUDE ALL ORIGINAL METADATA. This is very important as the metadata is used when the software aligns the photos (this is important in making a useful model).

Step 7

Once the custom preset has been made, you will need to export the desired photos using it. Highlight all the desired photos, by clicking the first image in the set and while holding shift click the last image in the set (you can unselect individual photos while holding CTRL). Then use File → Export to → RAW to TIFF VARLab, which is demonstrated in the screenshot below.

Step 8 (kinda Step 7.1)

Now you just sit and wait for the job to finish, you should see it with its respective loading bar in this Export Progress Window.

Once the export is done, you should be able to locate the export folder called in the original folder where the photos are, the folder will be called TIFF.
