Step 1: Adding an Image for Editing

  • To get started, first open up a new paint.Net file

  • Then, use File → Open and navigate to where you have stored the downloaded Camera image file for this guide (see SharePoint link at top of this guide)

  • Once the file is open, the editor should appear similar to below:

Step 2: Using the Magic Wand and Lasso Select to Remove Background

  • Start by selecting the Magic Wand tool

  • Now click anywhere on the white background

    • The Magic Wand tool basically selects all pixels in a given area that is an average of a given color (in this case, white) based upon the Tolerance slider (next exntry) - with 0 being absolutely only a specific color and 100 being all colors (regardless of hue)

  • Sliding the Tolerance slider, or using the + or - buttons to the Left and Right, you can adjust how absolute the color selection of pixels are

  • Setting the slider to 25% will cause a large chunk of white pixels around the camera to become deselected

  • Setting the slider to 56% selects the vast majority of the background, but also selects some border parts of the camera that are of a similar hue to white (ie: grey or silver). But theres a way to actually use this to our advantage

  • Turning on the Lasso Select tool, you can quickly and efficiently manually Select and Deselect parts of the image that the Magic Wand either selected or missed, by simply clicking and dragging over the area to add or remove said area from the active selection (area you selected with the Magic Wand):

    • When wanting to ADD to the selected area, Hold CTRL + Click and Drag

    • When wanting to REMOVE from the selected area, Hold ALT + Click and Drag

    • Review the following video to see this in action

d78caf48ba3e2e8ae052d6a155318ede.mp4

  • Once you’re happy with your selected area, simply hit the Delete key to delete the background

  • The background should now be gone, and replaced with a light and dark grey checkerboard pattern (signifying there is a transparent background and nothing is there)

Step 3: Feathering the Edges to Blend with Background

  • Now that the background is removed, we can go in and ensure that the edges of the camera are blended with the background and any extra bits we missed in our selection in the previous step can also be removed

  • Start by turning on the Magic Wand tool and setting the tolerance to 0%

  • Click anywhere in the area of the removed background. As this is a fully transparent area, setting the Tolerance to 0% will ensure the whole removed background area is selected

  • Now invert the selection (Edit → Invert Selection) to flip the selection so that the whole Camera area is selected

  • Now zoom into the Camera and find a good edge spot (most edges will suffice)

    • In the previous step, I missed some stray pixels at the edge of the Camera Lens - this is a good spot to use for the example

    • You want to make sure you are zoomed in on an edge so that you can see the Feather tools effects up close and can fine tune the amount of the effects application

  • Adjust the slider until a sufficient amount of the edge of Selected area is removed

Step 4: Applying Sharp and Gaussian Blur to enhance and cleanup Image

Using a combination of Sharp and Gaussian Blur effects, you can enhance your image to bring out subtle details and then soften the resulting image to clear up imperfections

  • Start by selecting the full image (CTRL+A)

  • Go to Effects → Photo → Sharp

  • Adjust the sliders to sharpen the image details similar to above

  • Now go to Effects → Blur → Gaussian Blur

  • Adjust the sliders to blur the image details similar to above so as to soften any over-sharped areas

Step 5: Squaring the Image

Its important to square images for rendering so that images will render correctly

  • Start by magic wand selecting the Alpha area around the camera with tolerance set to 0

  • Press CTRL+i to invert the selection so that the Camera becomes the selected area

  • Go to Image → Crop To Selection

  • The Camera should now fill the frame and fill the Canvas

  • Note the Image Size at the bottom of the screen - this needs to be evened out to make the image square

  • Go to Image → Canvas Size

  • Under the Anchor Section, click on the Middle Square of the 3x3 Square Grid - this will make it so that the Canvas will be resized from the center of the Image

  • Lets set the height and width of the image to an even 1700 - this will create a little alpha padding around the Camera and square the image nicely

  • The squared up image

Step 6: Resizing Image to a “Power of 2”

When it comes to texture rendering, it is very important to have the Image size be a “Power of 2” - this means that the image is one of the following sizes:

  • 8x8

  • 16x16

  • 32x32

  • 64x64

  • 128x128

  • 256x256

  • 512x512

  • 1024x1024

  • 2048x2048

  • 4096x4096

  • And so on (increasing by a power of 2 each subsequent size)

Since the Camera Image is currently 1700x1700, lets set it to a size of 1024x1024

  • Go to Image → Resize

  • Set the image Width and Height to 1024x1024, Set your Resampling to be Bicubic (Smooth) if not set by default

    • Resampling is an algorithm setting that governs the sizing and shaping of pixels when resizing the image. “Bicubic (Smooth)” and “Nearest Neighbor” are the most useful settings

  • Resized Image

Step 7: Saving the Image

Simple use File → Save As to save your image and set it to PNG type - PNGs are cleaner in image quality compared to JPEGs, and also store Alpha (transparent) layer information while JPEG’s are not capable of doing so
