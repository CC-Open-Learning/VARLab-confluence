VARLab : Bun (Burger?)

Created by User 96d6e on Jan 23, 2025

800 Triangles → 220 Triangles

Original Model (Agisoft/Photogrammetry Merged Model)

The goal of this experiment was still to get the triangle count down further. I followed the same process that I used for the pork shoulder and was able to create a lower-poly model successfully.

The only visible problem with the higher-poly model was that there is a visible seam from where the top side and the bottom side of the bun were merged together.

Results (High-Poly to Low-Poly)

Through creating a lower-poly model I was able to get rid of the visible seam where the two sides of the bun met. Although the lower-poly bun is visibly less round, I believe it looks more seamless and maintained it’s rich texture.

Issues I Ran Into

When I was first baking the texture from the high-poly model to the low-poly model the texture was coming up black due to a lack of data or not being able to find the data. In order to fix this I made sure that both Principle BDSF shaders were set to be Diffuse BDSF shaders, which allowed for Blender to map the texture correctly.
