Feature List

  1. Search for assets by a given name

  2. See list of search results (card view)

    1. Prioritize assets from the varlab library

  3. Filter results by various attributes (listed above)

  4. Sort results by various attributes (listed above)

  5. View information about an asset (model view)

  6. Compare multiple assets

    1. Be able to add multiple assets which needs to be compared when clicked on “+” button

    2. Selected assets will be compared side by side on a new page with all the details.

    3. The selected items will have a “X” button on the top which will remove them from the selected list.

    4. There will be a compare button on the sidebar which will take us to compare page.

    5. All the selected items will be shown on the sidebar.

  7. Bookmarks???? (This is stretchy/ambitious)

    1. Save assets to bookmarks

    2. Search Bookmarked assets?

    3. Remove assets from bookmarks

    4. View bookmarks in large view (like search results)
