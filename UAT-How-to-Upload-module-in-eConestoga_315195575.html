VARLab : UAT-How to Upload module in eConestoga

Created by Janani Sathyanarayanan, last modified by Stacey Dineen on Nov 26, 2021

Go to the eConestoga course shell that will be used for UAT. Add a module and call it [Project Name] - UAT. In this example, you would use “Hydraulics - UAT”.

Once you have added the module, click on the module topic in the left-hand Table of Contents menu.

Click on the Upload/Create drop down and select “New SCORM/xAPI Object

Click on Bulk Upload

Browse and select the SCORM zip folder and select Open. It will take a while to upload, there is a progress bar.

Update the name and description for the SCORM package (this is the name that UAT volunteers will see, so it needs to be user-friendly) and click Next

Select the uploaded file

Where it asks “Do you want a grade item created for this instance?” you must answer NO.

The content will be added in eConestoga

How to publish in SCORM

Log into VTS editor

Click on the import project and select the .vtp files from the respo. Folder and open

Double click on the module

Once the module is opened click on the Publish

Click SCORM/Web at the right

Toggle Web with SCORM 1.2 and click next

Then click on Publish

Save in the folder

A zip folder will be saved and the same can be exported in eConestoga
