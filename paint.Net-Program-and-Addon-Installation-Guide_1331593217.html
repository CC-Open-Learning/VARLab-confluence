VARLab : paint.Net Program and Addon Installation Guide

Created by Ian Kennaley, last modified on Mar 27, 2025

This article is in draft

Foreword: Purpose of this document is to guide you through the basic installation steps for paint.NET and how to download and install the Labs library of addons from SharePoint.

Note that all addons are free ones and are safe to share.

paint.Net is a free and open source image editing program that is a very effective alternative to Adobe’s Photoshop. Having been around for nearly 20 years, several fantastic free addons have since released that brings even more parity with the capabilities of PhotoShop. Interestingly, paint.Nets compression algorithm achieves between 40%-70% better compression than that of PhotoShop.

  • Step 1: Download paint.NET
  • Step 2: Unzip the file and Run the installer (Alternative: Run the installer directly from in the ZIP)
  • Step 3: Download the addon library from SharePoint and copy the files to the paint.Net Effects directory
  • Step 4: Open up paint.Net and verify that it installed correctly and that the addons have been added correctly
  • Additional Notes

Step 1: Download paint.NET

Go to the following link and click on the “Download Now - dotPDN” to download the latest version of the App

https://www.getpaint.net/download.html#download

Note: Make sure to click on the one with Free next to it. paint.Net is free and open



On the following page click on the “Free Download Now:” link to download the latest version of paint.NET (at time of this guides writing, the latest version is paint.NET 5.1.7)

Step 2: Unzip the file and Run the installer (Alternative: Run the installer directly from in the ZIP)

  • Unzip the downloaded installer and run said installer OR run said installer from directly in the Zip folder (Windows 11+)

  • Recommend choosing Express and then simply click through the prompts and wait for the installer to complete

  • Once finished, click the “Finish” button when prompted

Step 3: Download the addon library from SharePoint and copy the files to the paint.Net Effects directory

  • Start by downloading the Labs collection of addons for paint.Net from the following SharePoint link: paint.Net Addon Library

    • Note that these are addons that are freely available from Plugins - Publishing ONLY! - Paint.NET Forum and have been put into assembled so that they can be mass downloaded and then copied collectively to where they need to go. On that note….

  • Unzip or Copy the files in the downloaded folder to your installed copy of paint.Net’s Effects directory to install the addons, with the exception of BoltBaitPackInstall610.exe → Run this exe as it installs a large group of addons by the creator of the same name (BoltBait)

    • Effects Director:

      • This PC → Local Disk(:C) → Program Files → paint.Net → Effects

Step 4: Open up paint.Net and verify that it installed correctly and that the addons have been added correctly

  • Open paint.Net and verify that it opened correctly

  • Click on the Adjustments drop down and verify it appears as such

  • Click on the Effects panel and confirm it appears as the following

  • It is worth exploring these drop downs further, however there are too many to cover in this guide and it is therefore recommended that you explore and experiment with the features under the above headings (as well as with the features of paint.Net, some of which will be covered in other guides)

Additional Notes

  • Atm there are no additional notes, but as paint.Net develops (and documentation on the program naturally evolves) then more notes may be added here
