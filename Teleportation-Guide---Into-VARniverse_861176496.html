VARLab : Teleportation Guide - Into VARniverse

Created by Brodin Collins-Robb, last modified on Jan 27, 2024

Now that you have set up your teleportation out of VARniverse to your project in Teleportation Guide - Out of VARniverse, it's time to set up the teleportation from your project back to VARniverse!

 Instructions

To begin, you must have your project open in the VARniverse project (to open your project, navigate to the Mini-Games folder in the Project Assets folder and navigate through the semester and project folder to find your projects folder. Then, in your projects folder, find the scene file within your project and double-click the scene file).

  1. With your scene open, navigate from the Projects Assets window to VARniverse Assets → TeleportationStone → InfoCard. In this folder, there will be the script ‘returnToMain', which contains the following code:

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.SceneManagement;

public class returnToMain : MonoBehaviour
{
    // Update is called once per frame (this will allow that any point during your mini-game the player can return to this hardcoded main scene)
    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))//checks if you are pressing down the esc key which is the control to telepport
        {
            Debug.Log("interacted time to teleport");//debug log prints to the console for testing
            
            SceneManager.LoadScene("VARniverse");
        }
    }
}

This code will make it so that at any point during your project, the user can press the escape key to return to the main scene.

If you would instead set a different key to return to the main scene, then you can change 'KeyCode.Escape' on line 11 with any other key that you find best suits your project!

  1. In your project on your player or any other game object that exists in your scene for the entire duration of your running project, drag and drop the 'returnToMain' script from the folder described in step 1 into the inspector pane. (Remember to select the game object for the project hierarchy that you want to add the script to and drop the script under the existing components in the inspect (wait for the blue bar to appear where the script will be placed before dropping it into place))

At this point, when your scene is running, and you press the specified key from the returnToMain script, your scene should close, and the player should teleport back to VARniverse in the same position in which they left VARniverse and looking the same way!
