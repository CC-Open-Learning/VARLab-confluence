VARLab : Job Framework

Created by Alistair Campbell on Sept 20, 2024

Job Definition

A job should be any action that can be carried out by an NPC.

Job identification

Jobs should be identified by a custom script attribute, declaring them as an NPC job.

Job Assigning

What if the interactable provides what actions can be taken on it (in name only) and relevant parameters, and the NPC will contain the code for those actions.

Case: NPC Goes to box, checks if “take” is a valid action. NPC Takes, and checks what object to take from what the box contains.
