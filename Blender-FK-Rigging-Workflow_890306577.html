VARLab : Blender FK Rigging Workflow

Created by Ian Kennaley, last modified on Jan 04, 2024

THIS ARTICLE IS IN DRAFT

This will be a guide on generating a basic FK rig for use with models and a basic animation keyframing guide for creating a basic test animation to check for correct deformations.

Note 1: The following guide assumes you’ve completed modelling and texturing of an asset and therefore the following steps are purely related to Rigging.

Note 2: For the purposes of this section of the guide we will rig the Green KED Splint from the Ambulance and Paramedicine Simulation (in-development).

All Source Files for the Green KED Splint: Asset Files - Splint_GreenKED

Step 1: Body Bone Placement

Determine where you want your Body bone to be placed

  • With the above Green KED Splint, we’d want to place the Tail (bottom of bone) of the Body bone at the center between the top and bottom vertices of the bottom of the splint surface (furthest -Y point in the above screenshot) and the Head (top of bone) of the Body bone at the the center between the top and bottom vertices of the top of the splint surface (furthest +Y point in the above screenshot)

Step 2: Add Armature

In Object Mode add an Armature (Rig) to the scene, resulting in a single bone appearing at the center of the 3D scene

  • Hit Shift + A and select Armature

Step 3: Body Bone Tail Orb 3D Cursor Placement

With the Splint mesh selected, enter edit mode and select the top and bottom (z axis) vertices at the bottom of the splint body (furthest tip of -Y axis) and place your 3D cursor there

  • Edit Mode: Vertex Select Mode → [Select Vertices] → Right Click → Snap Vertices → Cursor to Selected

Step 4: Snap Body Bone Tail to 3D Cursor

Exit Edit Mode, Select the Armature Rig and Snap Selection to 3D Cursor

  • Bone Selected → Right Click → Snap → Selection to Cursor

Step 5: Body Bone Head Orb 3D Cursor Placement

Select the Splint Body mesh, enter edit mode and select the top and bottom (z axis) vertices at the top of the splint body (furthest tip of -Y axis) and place your 3D cursor there

  • Edit Mode: Vertex Select Mode → [Select Vertices] → Right Click → Snap Vertices → Cursor to Selected

Step 6: Select Armature and enter Edit Mode

Exit Edit Mode, Select the Armature Rig and Enter Edit Mode

Step 7: Align Head Orb to 3D Cursor

Select the Head of the Bone and Snap the Head to the 3D Cursor to finalize the positioning of the Body Bone

  • Select Head orb → Right Click → Snap → Selection to Cursor

Step 8: Rename Body Bone

Rename the Bone as bone_Body in Edit mode

  • With Bone Selected → Space Bar (Search Menu) → Type rename → Select “Edit → Rename Active Item” → Type in “bone_Body” → Hit Enter to Finalize

Step 9: Duplicate Body Bone for Pillow Bone

Duplicate the Body bone and then use the same 3D cursor process to position it to overlay the Pillow section of the Splint Mesh

  • Duplicate Body Bone and Position for better Placement: With Body bone selected → Shift + D → Right Click (snap back to origin) → Press Z → Type .3 → Bone will Be above Body bone → Press Space Bar → Type Rename → Select Rename Active Item → Type in “bone_Pillow” → Hit Enter to Finalize

  • Note that the Bone moved as it would as if it was a separate object in Object mode - this is because the “Tail” of the bone is considered the Pivot point of the bone and all transforms are performed from there (where pivoting and movement is concerned)

Step 10: Place 3D Cursor for first Bottom Right Flap Bone

Start setting up the right side flap bones by placing the 3D cursor at the first edge of the 3 loop bevels that set the topology up for folding, and then duplicate the Pillow bone and then repeat the snap process so that it is positioned at the 3D Cursor, and then rename the bone as “bone_Flap_Bottom_Right.000” and lastly position the Head orb of the bone to be placed at the next edge of the fold lines using the Snap function and then straighten the bone out by activating the Scale tool with the S key and hit the Y axis to constrain it to the Y axis and then type 0 and hit enter to straighten the Bone

Step 11: Place next Bone in Chain

For the next bone in the chain, place the 3D cursor at the middle of the next set of Fold edges, select the head of the first bone in the chain, extrude it with the E key and hit the X key to constrain it to the X axis and move it over partially towards the 3D cursor and then hit Enter to place it, change the Transform Pivot Point from Bounding Box to 3D cursor, and then activate the Scale tool with the S key and then hit the X key to constrain it to the X axis and then hit 0 to snap the bone Head orb to be in line with the 3D cursor

Step 12: Repeat Step 11 for next Fold Line

Repeat Step 11’s process for the next Edge Line of the current Fold Line section

Step 13: Final Bone in Chain Placement

For the last bone in this chain, repeat the same process as the last step, using the center Top and Bottom vertices of the far right edge of the Splint Mesh as the 3D cursor alignment point

Step 14: Top Right Flap Bone Placement with 3D Cursor

Next set up the right side top flap Bone structure by placing the 3D cursor in the center of the Top Flap Fold Lines, then duplicate the first two bones in the bottom flap chain (.000 and .001), rename these as bone_Flap_Top_Right.000 and bone_Flap_Top_Right.001 and then Snap these to the 3D Cursor. Once in place, place the 3D cursor on the furthest right edge of the Mesh and then scale the Head orb of the second bone on the X axis to align it with the side of the Mesh

Step 15: Mirror Right Side Bones onto the Left

Now that the Right side Flap Bones have been properly set up, set up the Left Bones by Snapping the 3D Cursor to World Origin, Set the Transform Pivot Point to 3D Cursor (if not already setup), go into Edit mode on the Rig and selecting all of the Right side Flap bones (bottom and top) Duplicate these, and then mirror them onto the left side of the Mesh by Scaling on the X axis and hitting -1 to flip them to the other side

Step 16: Rename Flipped Bones, correct their Serialization, and correct any Positioning Errors

Rename the flipped Bones to Left instead of Right (correcting the naming serialization and Right → Left orientation label), and then use the same 3D cursor placement process we’ve used up until now to correctly align the bone Heads and Tails to the corresponding Fold Lines so that everything lines up correctly

Step 17: Bone Parenting Setup

Now that all of the bones are properly placed, its important to set up the Bone Parenting relationships so that bones and their mesh elements will move around correctly when animating. For this rig, you’ll want to have the .000 bone on the top and bottom Left and Right flaps as well as the Pillow bone all be parented to the Body bone.

  • Parenting Process: [In Edit Mode] Select a Bone → Shift+Left Click Bone to be Parented to → Right Click → Parent → Make → Keep Offset

  • Note: There are times where Parenting with the Connected option is applicable - a good example are the Flap bones on this rig - due to the jointed nature of the Mesh where the folds are it makes sense to have these Bones connected together to form a chain.

  • Note: Instead of parenting Bones one at a time you can select multiple bones at the same time and perform the Parenting Operation to all of these bones simultaneously

Step 18: Separate Mesh Elements for Componentizing

In Object mode, select the Splint Mesh and then go into Edit mode, select all Mesh elements, and then separate out the Mesh by Loose Parts to prepare for the Weight Painting Process. Be sure to verify Mesh separation back in Object mode

Note: It’s very important, as part of the Mesh Optimization process, to Componentize the Mesh so that Elements aren’t necessarily joined up with geometry (a good process to eliminate un-necessary geometry)

  • Select all Mesh Elements in Edit Mode: Press the A key

  • To Separate by Loose Parts (Keybind): P Key → By Loose Parts

  • To Separate by Loose Parts (Mesh Dropdown): Mesh Tab → Separate → By Loose Parts

Step 19: Selectively Combine Separated Elements into Joined Components

Go through the separated Mesh elements and selectively combine elements based upon the corresponding bone the mesh element(s) will be Weight Painted to, and also rename the Armature to match the Rig Name Convention (in this case: rig_Splint_GreenKED)

  • Join Function (Right Click): [Select Mesh Elements] → Right Click → Join

  • Join Function (Keybind): Ctrl + J

Note 1: It is not necessary to rename the Mesh elements in this Step as those elements will be recombined together fully into 1 single Mesh after the Weight Painting Process

Note 2: Due to the folds on this particular mesh, to ensure that the Mesh folds deform smoothly smaller folding bones are needed to more finely control the fold sections for this purpose - thus, Mesh elements should only be Weight Painted to the larger bones only, with the exception of the body of the Splint Mesh as it has the actual Fold sections on it (and thus needs to be Weightpainted to all bones, save for the Pillow bone).

Step 20: Parent Mesh Elements to Armature with Empty Groups (Weights)

Begin the Weight Painting process by selecting all Mesh elements in Object Mode, then selecting the Rig, and then Right Clicking and Parenting the Mesh Elements to the Rig with Empty weights. Also confirm that Parenting was successful by verifying the Bone Vertex Groups in the Object Data Properties Panel

  • Parenting Workflow: [Select Mesh Elements] → [Select the Rig to make it the Active Selection] → Right Click → Parent → Armature Deform → With Empty Groups

Note 1: Parenting With Empty Groups basically sets the Mesh elements up with the Data References to the Rig Bones but ensures that there is no actual Weight Painting occurring during this process (ensuring all Meshes have 0 Weight painted to the bones).

Note 2: It is possible to use the “With Automatic Weights” to have Blender automatically set up the weights, but this often leads to a lot of messy weight painting and often requires additional work to clean up. Using Empty Groups instead ensures the Weight Painting process starts with a clean slate essentially. The reason this is important will become further clear in Step 21 and Step 22

Step 21: Weight Painting Process - Face Based

Select each Mesh element individually, then select the corresponding Bone it is to be Weighted to in the Object Data Properties panel, and then switch to Weight Paint Mode and using the Face Selection Mode, Set the Weight Strength to 1 and then Apply that Weight Strength fully so that the selected Mesh element will only be moved and deformed by that Bone (and no other Bone).

Important Note: Do Not perform this Weight Paint Process on the Mesh body element (the big piece) as there is a slightly different Weight Paint workflow to be done for this piece that will be completed in Step 22

Important Note: Blenders Weight Painter utilizes a Heat Map visualization of the strength of elements weighted painting to bones - Blue is what appears when there is 0 Strength (or Cold on the Heat Map) and progresses through the Color Spectrum based on the level of Strength progressing up to a value of 1 for Strength that is visualized as solid Red (or Hot on the Heat Map). Specific Color order is Blue → Aqua → Green → Yellow → Red

Important Note: Note that when in Face Select mode, when Unselected Faces appear Blue with a Less Saturated hue to denote that they are are Unselected

Important Note: Note that when in Face Select mode, when Selected Faces again appear full Blue to denote that they are are Selected

Process Note: This is the full basic process of Weight Painting by Setting Face Weight and Strength to full through the Set Weight function. Do note that you can also perform the same function via the Set Weight function Keybinding which is Ctrl + K.

Switch back to Object mode to repeat this process for the remaining Mesh elements and the corresponding large Bones. Reminder: Do Not use this process for the Body Mesh Element as that will be completed in the next Step (22).

For referencing screenshots will be provided of the Mesh Elements selection with the needed bone switched to, the Unweighted view, and the Fully Weighted view - however this will not be as indepth as the above (as the process is now fully detailed)

Step 22: Weight Painting Process - Vertex Based

As noted earlier, the Splint Body mesh requires a slightly different process for the Weight Painting due to the Folds added to the Mesh Geometry for the Animation Deforming process. As such, this workflow involves switching between Edit Mode and Weight Paint Mode so that the needed Vertex 's can be more easily targetted to the Bones they were aligned to earlier and the Weight Painters Vertex Selection mode can then use the Set Weight function to target those vertices → Switching to each corresponding Bone the Vertexes are to be Weighted to in the Object Data Properties panel while in Edit Mode, and then switching to Weight Paint Mode and using the Vertex Selection Mode, Set the Weight Strength to 1 and then Apply that Weight Strength fully so that vertices will more cleanly deform by that Bone (and no other Bone) - and then switching back to Edit Mode and repeating the process for each Bone.

For the purposes of keeping this section of the guide as concise as possible screenshots will be provided detailing the flow however the process is basically the same as when using Face Selection Weight Painting (detailed in Step 21) so follow the above Step instructions (22) and reference the following screenshots for the workflow

Important Note: As we are applying the Weight directly to these Vertices, there is some automatic Falloff applied (Red → Yellow → Green → Blue) and this helps to smooth out the Deformation process further during Animation

Step 23: Recording an Animation Test to check Rig Deformations

Now that Weight Painting is complete, run a Pose Test to ensure that all Mesh elements deform correctly. To Animate, Select the Rig in Object Mode and then change from Object Mode to Pose Mode, Bring up a Timeline Window in the Editor view and configure the Animation Length and Playspeed, and then Perform a Transformation on the selected Bone(s), record the movement as a Keyframe, and then Repeat this process until the Rig animates as desired.

  • To Pose: Select Rig in Object Mode → Switch to Pose Mode → Select Bone → Move/Rotate as needed → Insert Keyframe

Important Note 1: The above process is a basic how-to for recording animations with bones - you can perform actions like Rotating, Scaling, and Position transforms to pose the Rig.

Important Note 2: Once you’re sure all of the deformations occur correctly, be sure to delete the keyframes you just created for testing so that you can record required animations from a blank slate.

Step 24: Add Root Bone and Adjust Weights as needed

Now that we have everything deforming correctly, lets finalize the Rig by adding in a Root bone for running animations. Set the 3D Cursor to world origin (center) and add a new Bone and reposition the Head of the bone to the top layer of the Splint mesh body. Rename this bone as bone_Root and then parent the Body bone to this Root bone to finalize the Rig.

Step 25: Combine Mesh Elements into a Joined Mesh for Animating and Final Export

Once you have confirmed that all elements deform correctly and created the basic Root bone, go back to Object Mode and select all Mesh elements and combine them back together into 1 single mesh (instead of multiple separate Meshes). The Weight Painting performed in the previous steps should be retained and combined into the Recombined Mesh’s Vertex Group Data, meaning that the Mesh will add the new bone reference and not wipe out any of the existing Weight Painting data.
