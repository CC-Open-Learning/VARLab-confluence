VARLab : Classifying Object Geometry & Texture

Created by estronge, last modified by User 96d6e on Jul 03, 2024

If any rigging is needed, it is recommended that an artist creates the model/asset.

Texture

Geometry

Simple

  • Simple pattern

  • Not too many colours

  • Not shiny or transparent 

  • The shape is describable using a basic 3d shape

Complex

  • Patterned 

  • Multiple textures/ materials 

  • Shiny

  • Transparent 

  • The object cannot be described using a single basic 3d shape 

  • Multiple parts/components

  • Lots of surface details (bumps, folds, etc…)
