Below is a breakdown of VR control in VARniverse…

Each description in the table corresponds to a controller action shown on the controllers above.

Note: There are two ways to control movement in VARniverse the first uses right right controller for fine control movements but can cause motion sickness.

Use the left-hand controller, which are snap movements that help avoid motion sickness.
