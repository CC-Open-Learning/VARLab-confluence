API Things

These API calls have been created by analyzing network traffic on the sites while performing various actions. We need to make sure that we are good to use them before we depend on this, but these calls are very powerful and we will be able to serve much better information by using them.

API calls must be made through a cors-anywhere server on port 27000.

This is the minification/standardization system for Sketchfab that should be the standard for all going forwards

            let miniResult = {
                name: result.name,
                price: result.price/100,
                thumbnails: result.thumbnails,
                storePage: 'https://sketchfab.com/3d-models/' + result.uid,
                store: 'sketchfab',
                animated: !!result.animationCount
            }

Sketchfab

https://sketchfab.com/i/search?count=24&in_store=1&processing_status=succeeded&q=car&restricted=1&sort_by=-relevance&type=models&cursor=bz05Ng%3D%3D'

Params

  • count = number of items to serve

  • in_store = ???

  • processing_status = ???

  • q = search term

  • restricted = include nsfw content (1 = yes I think)

  • sort_by = sorting system

    • Options: ‘relevance’, ‘-orderCount’ (best selling), ‘-rating’, ‘-publishedAt’ (newest), ‘publishedAt’ (oldest), ‘price’ (low to high), ‘-price’ (high to low)

  • max_face_count

  • min_face_count

  • licenses=783b685da9bf457d81e829fa283f3567 (this is the standard license that allows commercial use, because SF wants to make up their own licenses)

  • Formats

    • 3ds Max

      • file_formats=3ds

      • file_formats=max

    • file_formats=fbx

    • file_formats=blend

    • file_formats=obj

    • file_formats=c4d

    • Lightwave = file_formats=lwo

    • Maya

      • file_formats=ma

      • file_formats=mb

    • Unity

      • file_formats=unitypackage

    • stl = file_formats=stl

    • Sketchup = file_formats=skp

    • Modo = file_formats=lxo

    • polygon? = file_formats=ply

    • colladata = file_formats=dae

    • Unreal

      • file_formats=uasset

Sketchfab uses SSR for their product pages, scraping data is the only way to really gather it from there.

Getting all of the model information from viewing a model page is under a div with the class c-model-3d-information__col, and all the information total is under the div with the classstore-informations.

CGTrader

https://www.cgtrader.com/search.json?page=1&keywords=car

Fully filtered

https://www.cgtrader.com/search?animated=1&rigged=1&keywords=car&price_range[]=range_20_1000&polygon_range[]=range_10_1000000

  • page = pagination page. We may need to integrate a pagination system into our search to continue to generate results in case the initial search is not sufficient

  • keywords = The search term

  • Animated = (1 for true, dont include for false)

  • Rigged = (1 for true, dont include for false)

  • price_range[]=range_X_Y = X is lower limit, Y is upper limit (default upper is 1000)

  • polygon_range[]=range_X_Y = same as above, default upper is 1000000

  • File Formats

    • FBX = file_types[]=5

    • OBJ = file_types[]=12

    • STL = file_types[]=51

    • MAX = file_types[]=17

    • blend = file_types[]=21

    • c4d = file_types[]=6

    • maya = file_types[]=11

    • glb = file_types[]=117

ArtStation

Artstation has no API to hook into and they do all of their document rendering on the server. The only way to pull data from them would be to build a web scraper into our search engine. This is feasible because ArtStation does not use generated class names.

Unity Asset Store

Unity uses GraphQL. This makes it a lot more complicated to pull data from their store. The API returns readable and useful values, but it is much harder to query.

3DModels.org

Data can be scraped out of the response to the request

https://3dmodels.org/page/3/?s=car

Known Issues

  • Sketchfab thumbnail images are not always a consistent size. usually 300x150, but sometimes are considerably smaller than that

  • Sketchfab delivers 10 results on the search endpoint with their developer API. We can increase this number by using the API call above

  • Sketchfab Developer API does not provide the price of models when doing a search. (again, the one above doesn’t have this issue)
