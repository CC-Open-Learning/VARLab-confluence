VARLab : VARniverse Contents

Created by Brodin Collins-Robb, last modified on Jan 27, 2024

Below is a breakdown of all the main components in VARniverse…

Each number in the table below corresponds to an object in the VARniverse scene image above. See the column to the right of the number to get more information on what each item is.

1

UI Hub World

This is the introduction UI text that you see at the start of exploring VARniverse.

2

Main Player

This is the player that you explore the VARniverse world with, this is how you look around, walk, jump and view the world!

3

Environment

This is a game object folder that holds all of the environment scene assets for VARniverse. Whenver you add a game object to the scene it will be added to the hierarchy and this is way to keep it organized. So assets that are a part of the VARniverse environment all go to this folder and can even be organized withing like Village assets, or Science Building assets, etc.

4

MiniGame-Teleportation-Stones

Like the Environment game object folder this holds all of the teleportation stones within VARniverse. Since these stones are interactable and are not necessarily part of the environment they get their own folder to make adding and finding the teleportation stones easier.

5

Imported Assets

Now into the Projects Assets folder… This is the imported assets folder than if you were to add assets from the https://assetstore.unity.com/ you can important premade assets to use in VARniverse or your mini-game. Within this folder there will be folders for each asset pack and once again this helps keep the project organized.

6

Mini-Games

This is the Mini-Games folder that holds all of the mini-game projects in VARniverse. These mini-games are subdivided by semester and mini-game folders so each project gets its own folder in the VARniverse world to keep all of the projects contents organized.

7

Scenes

This is the scenes folder that holds the VARniverse scene and scene information like lighting maps.

8

Scripts

This is the scripts folder that holds all scripts used in VARniverse. This helps keep things organized and easy to find with the added benefit of having one spot for all of the ‘the code’.

9

VARniverse Assets

This is the assets folder specifically made for VARniverse object/prefabs. This folder holds items that are specifically designed for VARniverse like the teleporation stones files and prefabs. This folder also contains README folder with text instructions are getting started with VARniverse details. These pages in Unity reflect the instruction pages in confluence.
