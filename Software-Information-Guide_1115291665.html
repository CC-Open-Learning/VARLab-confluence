3DF Zephyr

Standalone Software Application

Pros:

  • Free for lite version

  • Lightweight

  • Simple interface

Cons:

  • Lite version implies missing functionality, may not be an issue

  • Lesser processing capacity and speed than competitors

Free (Although you can also request a 30 day free trial?..)

  • Asset generation for video games

  • Medical VR Simulations, etc..

PLY, STL, OBJ/MTL

Agisoft Metashape

Standalone Software Application

Pros:

  • Photogrammetric triangulation

  • Dense point clouds

  • Editing (I have not seen this present in any of the inexpensive software)

  • Classification

  • Most recommendations I have found point to either this or RealityCapture for quality, speed, price, features

Cons:

  • Expensive

  • More complex than our use cases

Various licenses (Recommend investigating Floating or Educational License)

One-time fee:

Standard: $179 (still quite complete)

Pro: $3499

  • Research

  • Defense

  • Security

  • Surveying

  • Transportation

  • Mining and quarrying

  • Utilities

  • 3D design

  • Agriculture and Forestry

  • (More than applicable, probably overkill)

COLMAP

GitHub repo

Pros:

  • Free to use and just requires a citation

  • Many different platform options

  • Extensive documentation

  • Open source

Cons:

  • A source said that processing time is 15-20 hours per image.. (thought that is currently unsubstantiated)

Free

  • Structure-from-Motion (SfM) and Multi-View Stereo (MVS) pipeline

  • Visual reconstruction of ordered and unordered image collections

TXT (not really sure if it is possible to convert to one of our desired formats as far as I can tell but still need to confirm)

Kiri Engine

Mobile Application (Android)

Pros:

  • Mobile so image capture for outdoor environments is more painless

  • Can export in .fbx

  • Tutorials exist for Unity Integration

  • Adjustable texture and mesh quality export settings (Most of the software listed likely have this, but since I actually tested this one, I thought it would be good to note somewhere)

  • Paid version has a lot of desirable features but as it stands, the outputs I got were not good enough to be used even as a generic, non-interactable asset

Cons:

  • Takes a while to process in free version (You have to wait in a queue) So trusting this for anything timely may not be a good idea

  • Free version has maximum of 70 photos for each output

Free or Paid ($48-78 annual per license)

  • Reconstruct 3D models of objects in photographs

OBJ, FBX, STL, GLB, GLTF, USDZ, PLY, and XYZ

Meshroom

Standalone Software Application

Pros:

  • Open source

  • Lightweight

  • Simple to use

  • Free

  • Good at producing images from a very rough textured object

Cons:

  • Potentially unforgiving when trying to capture plain, or shiny object and may not work

  • May require tweaking settings to produce better output

  • Requires more dedication on the users part

Free

  • Reconstruct 3D models of objects in photographs

OBJ (Need to verify if that is the only option)

Cons:

Reality Capture

Standalone Software Application

Pros:

  • Extensive

  • Used for games (primarily in UE but also in Unity)

  • Free for educational purposes

  • Very good performance

Cons:

  • Free version has a generation limit of 50 photos per generation, which means less sparsely populated/detailed point clouds and therefore a less clear image

Free to use (Need to verify)

or starting at 1,250 USD/year for more advanced subscriptions

https://www.capturingreality.com/Industry-UseCases

Data sheet: here

VisualSFM

GUI/Command line application

Pros:

  • Runs fast by exploiting multicore parallelism for feature detection, feature matching, and bundle adjustment

  • Reconstruction process happens in real-time and thus gives a better idea of what is actually happening within the process

  • Seems better at generating layout of environment than actually creating visually usable assets

Cons:

  • Doesn’t seem to support usage within the Unity Engine

  • Generates output in .nvm

  • Seems very specialized (possibly too special to accommodate any use cases for us, but will continue to research)

Free for personal, non-profit or academic endeavors

  • 3d Visual Reconstruction

NVM (Note: I have not yet found any way to convert a file of this type into a usable format for us)
