VARLab : Software Installation

Created by Jordan Robinson, last modified by Jonathan Bezeau on Aug 14, 2023

Below is a list of software currently used at the VARLab to develop in Unity on the CORE Framework project and Digital Learning Experiences. If you would like to use a different program for an IDE, text editor, or Git client you are free to do so.

Visual Studio

For programming our components in Unity, we are primarily using Microsoft Visual Studio. All students with a Conestoga email address have access to a serial key for Visual Studio Enterprise through Microsoft Azure.

While installing, make sure to add the Game development with Unity plugin.

VSCode

VSCode is a handy tool for programming shaders in Unity. With the Shader Languages Support and ShaderLabFormatter plugins, you get both syntax highlighting and formatting for all .shader files.

SourceTree

For our Git client we are currently using SourceTree, for information on how to set up SourceTree with the CORE Engine project, please visit Setting Up Bitbucket & Sourcetree

Unity

Install the latest Unity version (Unity 2021.3.24f1 as of 08 May 2023), either through the Unity Download Archive or from the Unity Hub desktop application. Select a personal or student license.

Microsoft Teams

For almost all of our communication, we are currently using Microsoft Teams.

If you cannot log into the VARLab Teams organization with your Conestoga email address and need an invite, let us know!
