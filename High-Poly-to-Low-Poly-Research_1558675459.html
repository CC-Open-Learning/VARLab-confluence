VARLab : High-Poly to Low-Poly Research

Created by User 96d6e, last modified on Jan 23, 2025

This document is currently research-based and not necessarily a tried-and-true method. It will explore turning a high-poly model (either through scanning or photogrammetry) into a low-poly model through texture baking.

Having optimized models is essential for our DLXs, as they require low-poly models to function properly across various computers at different performance rates. It also helps to ensure that the DLXs can be provided in VR format.

3D scanning and photogrammetry provide great textures as they are photorealistic and typically have realistic and even lighting. Our goal is to be able to use these textures and apply them to low-poly models. This process will make the models usable in situations where performance is paramount while still providing rich textures and environments.

This process uses techniques including (but not limited to) UV unwrapping through smart projection, baking settings, and creating low-poly meshes. Creating the low-poly mesh will start with a loosely accurate handmade mesh and then use modifiers such as shrink-wrap.

I explored other methods to try and streamline this process and will also describe those methods in this document.

Pork Shoulder Experiment (Tray Flipped Model)

The Best Method So Far

Overview

  1. Start with a plane and subdivide it around four times depending on the level of detail required for the model (you can always add more edge loops later)

  2. Shape the plane to the object from a top-down view (I modelled the plane for the bottom of the model first)

    1. Get it close enough and try to keep polygon quads rather than triangles (even if they almost look like triangles), as it makes it easier to get a smoother surface, especially when applying modifiers

  3. Once the first plane is close enough to the original models shape, you can duplicate it and change the shape to match the top of the model better (this step might not be necessary)

    1. If you alter the shape, try to keep the same number of edges on the outside loop as it makes it easier to connect the two separate planes in the future)

  4. Shape both the top plane and the bottom plane to match the height topology of the model; this is done from the side views

  5. Connect/join the two separate planes into one object

  6. Using vertices mode, select the vertices you want to fill (two from the top plane, two from the bottom plane) and fill them to create a joining face

    1. Repeat this process around the model until both planes are completely connected

  7. Create multiple edge loops around the model (across all the new faces); this will create more room for the geometry to bend around the model

  8. Do your best to shape the rest of the handmade model around the original high poly model accordingly

  9. Once the model is roughly in the right shape, you can apply the shrink-wrap modifier

    1. This will snap the handmade model's geometry to more closely resemble the high-poly model's geometry without adding more faces

    2. Also, make sure that you apply the modifier

  10. I then relax the vertices to create a more fluid and organic shape

  11. Go through the geometry and clean up any weird extrusions or unnecessary faces that were created/shifted when the modifier was added and applied

    1. Ex: multiple small faces by the circumference could be combined to decrease the amount of small faces

    2. Ex: there might also be some places where the modifier did its best to fit the geometry but created a sharper ridge, try to relax anything that doesn't resemble the original models shape

  12. Select the entirety of the model and UV Smart Project it to unwrap it, separate the islands by setting the margin to 0.010m

  13. Shade smooth both the low-poly model and the high-poly model

  14. Now, we can start the texture-baking process

  15. Create a new material for the new models texture, switch the BDSF to a Diffuse BDSF

  16. Then, add a new image texture node

    1. Create a new image by selecting the new button and creating a new file with an appropriate name; make sure the suffix is "_Color" to make it easy to find

    2. Connect the image node to the BSDF colour-to-colour

  17. Go into the Render Window from the toolbar

    1. Switch the Render Engine to Cycles

    2. Bring the Render Max Samples down to 128

    3. Go into the Bake dropdown

      1. Switch the Bake Type to Diffuse

      2. In the Influence dropdown, have only the Color checkbox selected in the Contributions

      3. Check the Selected to Active checkbox and set the Extrusion distance to 0.5m

      4. All the other settings should remain the same

  18. Select the new low-poly model, select the image texture node in the shader editor

  19. Then select the high-poly model in the data outliner (layers or object file viewer) hold Ctrl, and select the new low-poly handmade model

  20. Then hit the Bake button in the Bake window

    1. A loading bar should appear at the bottom of the screen

  21. Once the image texture has been created, you should see it now on the low-poly model

  22. Make sure you save the new image!!! You can do this in the UV editor window through Image and then Save as

  23. Yay :)

Detailed Description with Images

Creating the Planes

I first started with a plane for both the top and a separate plane for the bottom of the pork shoulder. I shaped the first plane to the top of the model and then duplicated it so that only minor adjustments were needed to shape it to the bottom. I then rotated the model using the numpad and the x/y/z axis gizmo to shape the height of the planes to match the model better.

The goal of creating a rough shape around the model is not to be perfect, but to get it close enough to where the shrink-wrap modifier creates more uniform and clean geometry/topology.

Connecting the Top and Bottom Planes

To connect the top and bottom planes, I chose two vertices from the top plane and two from the bottom plane that approximately lined up over each other and then filled them to create a new face. I repeated this process until all vertices were connected to create an airtight model.

I then added an edge loop around the model in the center of the height and around five edge loops both above and below the original edge loop around the model. This created much more geometry that would also help maintain clean topology down the line.

I didn't spend too much time trying to match the sides of the low-poly model I created to the high-poly model. I did bring out the sides slightly to match the curvature but didn't alter too much.

Applying the Shrink-wrap Modifiers and More

Apply the shrink-wrap modifier and pray.

Editing Some Topology Issues

I first corrected some topology issues by merging vertices by distance to ensure that there were no duplicates or extra vertices and also potentially got rid of some tiny faces. I also applied a relax vertices effect to make the model look more organic; it also assists in creating uniform polygons and topology. I then went around the model and checked for any sharp or weird-looking edges to try to flatten them; this also included getting rid of extremely squished faces by merging them. There was an edge around the model that was a bit harsh and sharp so I manually beveled that out to make it look more natural. I didn't spend too much time trying to create something perfect, but rather just enough time to make it "close enough".

Render Setup

Before I could do any texture baking there were a few extra steps that would make all the difference in creating a successful bake, these include:

UV Smart Project

Shade Smooth

Both can be found within various menu systems, but they ensure that the texture can map itself correctly to the model and that the low-poly geometry won't interfere with the texture and create more geometric separations in the texture map.

I also took another step to ensure a smooth and correct baking process; I recalculated all the outside normals to ensure correct face orientation.

Things I Learned Along the Way

There is such a thing as too little in terms of geometry. I tried to do under 2000 triangles and found it difficult to bake the original texture onto the new model without much distortion.

Adding more edge loops and places for the geometry to bend to the original high-poly scan helped create a more uniform topology that better reflected the shape of the high-poly scan. It also helped manipulate the low-poly shape to fit closer to the high-poly scan so that the modifier had an easier time getting the low-poly model to the correct shape.

Other Attempted Methods

I tried to decimate the original model using the collapse method, and not only did it almost blow up my computer, but it also only got it down to 50k before the texture baking stopped working properly. This could be something to look into, though.
