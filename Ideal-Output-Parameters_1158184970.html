VARLab : Ideal Output Parameters

Created by estronge, last modified on Jun 06, 2024

This page outlines the specifications for an acceptable asset created through the 3D scanning process.

Overall Recommendations

These are general guidelines that unless otherwise specified, can be safely assumed as the standard.

  • Between 512 x 512 or 1024 x 1024 for texture mapsÂ 

  • Material Type: PBR, MRO PBR, Cel Shade, etc (Other variety would be noted)

  • Export to fbx or obj (See Output File Conversion Table if you run into trouble)

  • Topology has to be fairly clean

  • Manifold geometry

Nice to haves:

These are extra guidelines that are beneficial, but not required by default

  • Exports of GLB's due to it being useful in other applications and include the textures.

  • Collada is another potentially useful format type

  • Production Files for Substance Painter and Blender/Maya/3DS Max
