First Steps

While there is a lot of information to take in, your first steps should ideally be:

  1. Log into Office 365 using your work account.

    1. Install Microsoft Teams.

    2. Specify your work hours as a Teams status message if you are working part-time.

  2. Complete the mandatory employee training modules for Conestoga College.

  3. Log in to your Atlassian accounts.

    1. Jira

    2. Confluence

    3. Bitbucket

  4. Install software outlined in Software Installation.

  5. Configure your source control tools with Setting Up Bitbucket & Sourcetree.

  6. Synchronize the asset repository by checking out Using the File Share.

  7. Get acquainted with the Jira, Confluence, and Bitbucket spaces linked above and review Using Jira.

  8. Add yourself to the Team Roster by following How To Fill Out The Team Roster.

  9. If you are new to Unity, read through Getting Started With Unity.

  10. Begin working on the Adventure Game Onboarding Project!

  11. Show the presentation on how to use the service portal.

If at any point you get stuck or need clarification, please don’t be afraid to ask for help!

Feel free to ask questions on the Onboarding group chat in Teams if you want everyone’s insight into a problem.
