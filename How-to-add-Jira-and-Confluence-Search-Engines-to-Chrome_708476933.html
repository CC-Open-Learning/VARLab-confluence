VARLab : How to add Jira and Confluence Search Engines to Chrome

Created by vtummillo on May 16, 2023

This page will outline how to add Jira and Confluence search engines to Google Chrome, which can be accessed via the search bar using a designated shortcut. Please see below for detailed steps and screenshots on enabling this feature.

 Instructions

  1. Launch Chrome and navigate to Default Browser

2. Select Manage search engines and site search

3. Under Site Search, select Add

4. For the Search Engine and Shortcut field, these can be set to whatever you prefer. The URL with %s in place of query needs to include the following:

Confluence: https://varlab-dev.atlassian.net/wiki/search?text=%s

Jira: https://varlab-dev.atlassian.net/browse/XRML-19?jql=text%20~%20%22%s%22

5. When you enter a “c” or “j” in the search bar, followed by a space, you can search Confluence or Jira, respectively.
