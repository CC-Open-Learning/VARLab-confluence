https://github.com/crashkonijn/GOAP?tab=readme-ov-file

THIS IS BIG. Entire GOAP framework including editor tooling for action graph creation. This is basically everything I wanted to build. Open Source, totally free.

  • Spending some time working through this system, it is immensely complex and the documentation is somewhat outdated.

  • The investment for setup is very high. It may be best to learn from this and make a more streamlined system using the techniques applied here.

  • Uses complex scriptable object setup to configure actions. Scriptable Objects are a very editor friendly system of configuring jobs. This may be a good thing to apply to our own developments.

  • Docs https://goap.crashkonijn.com/
