Created by Ruwini Perera, last modified on Apr 04, 2025

Overview:

To improve the current manual testing workflow at Varlab, QAlity Plus – Test Management for Jira can be considered as a solution. This Jira plugin provides a streamlined and integrated approach to managing test cases directly within Jira.

With QAlity Plus, test steps can be defined directly inside a Jira issue, allowing the QA team to maintain everything in one place. The plugin also offers a clear display of testing results, which enhances visibility and traceability of test outcomes.

Key benefits include:

  • Ability to reuse test cases, reducing redundant effort.

  • Better organization and clarity in test execution and result tracking.

  • Seamless integration with Jira, eliminating the need to switch between multiple tools.

The following demo provides a step-by-step explanation of the entire test management process, including:
✅ Creating a folder structure to organize test cases efficiently.
✅ Writing and managing test cases within Jira.
✅ Setting up test cycles to group related test cases for execution.
✅ Executing test cases and tracking results in real-time.
✅ Generating reports and analytics to assess test coverage and identify defects.
